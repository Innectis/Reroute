-------
Reroute
-------
A mod for simpler minecart travel systems as a Bukkit plugin.

For netbeans:
	Put this repository in the src/ directory, and compile it as you would any other Bukkit plugin.

Wool is used for this mod:

Green is booster
Lime green is triggered booster
White is Decelerate
Yellow is Junction
Red acts like a stone pressure plate
Blue acts like a wooden pressure plate
